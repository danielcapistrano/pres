Spatial distribution and local integration of migrants in Ireland

Daniel Capistrano, Evan Carron-Kee, Lúcás Rohan et al.

October 2025

Intro

Objectives

  • Geographical distribution of migrants in Ireland

  • Characteristics of the communities where migrants live

  • Residential segregation across neighbourhoods

  • Factors associated with residential segregation

Context

Literature - Spatial distribution and integration

  • Integration as a spatialised process (Gilmartin and Dagg 2021);

  • Different paths to integration depending on origin (Heath and Schneider 2021);

  • Housing as marker and mean of integration (Ager and Strang 2008) and space for bridges, links and bonds;

Literature - Correlates

  • High concentration in areas with high share of privately rented accommodation (Fahey et al. 2019)

  • Housing tenure, affordability and availability (McGinnity et al. 2024; Lima 2025)

Previous report

  • Small decrease in foreign-born segregation between 2011 and 2016.

  • “We might expect greater residential clustering among more tightly defined groups that share linguistic or cultural characteristics.”

  • Chapter 1: Literature review
  • Chapter 2: Geographical distribution and segregation
  • Chapter 3: Characteristics of local communities
  • Chapter 4: Policy context and implications

import {Inputs} from "@observablehq/inputs"
import {Plot} from "@observablehq/plot"
viewof ui = {
  const container = html`<div style="font-family: sans-serif; max-width: 800px; display: flex; justify-content: space-between; align-items: flex-start;">
    <!-- Left Column for Sliders -->
    <div id="sliders-column" style="flex: 1; margin-right: 20px;">

      <div id="sliders" style="margin-bottom: 1em;"></div>

      <button id="calculate-btn" style="margin-top: 1em; padding: 0.5em 1em;">Calculate Dissimilarity Index</button>

      <div id="result" style="margin-top: 1.5em; font-weight: bold;"></div>
    </div>

    <!-- Right Column for Plot -->
    <div id="plot-column" style="flex: 1; display: flex; justify-content: center; align-items: center;">
      <div id="chart" style="width: 100%; max-width: 400px;"></div>
    </div>
  </div>`

  // Constants
  const AREAS = ["Small Area A", "Small Area B", "Small Area C"]
  const N_RESIDENTS = 100

  // Create sliders and store their elements
  const sliders = AREAS.map(area => {
    const label = html`<label style="display: block; margin-top: 1em;">${area}</label>`
    const input = Inputs.range([0, N_RESIDENTS], {value: 20, step: 5})
    container.querySelector("#sliders").appendChild(label)
    container.querySelector("#sliders").appendChild(input)
    return input
  })

  // Button click event
  container.querySelector("#calculate-btn").onclick = () => {
    const g1_counts = []
    const g2_counts = []
    let g1_total = 0
    let g2_total = 0

    for (const slider of sliders) {
      const g1 = slider.value
      const g2 = N_RESIDENTS - g1
      g1_counts.push(g1)
      g2_counts.push(g2)
      g1_total += g1
      g2_total += g2
    }

    const D = 0.5 * g1_counts.reduce((sum, g1, i) => {
      const g2 = g2_counts[i]
      return sum + Math.abs(g1 / g1_total - g2 / g2_total)
    }, 0)

    const percentD = +(D * 100).toFixed(1)
    container.querySelector("#result").textContent = `Dissimilarity Index: ${percentD}`

    // Draw the chart
    const plot = Plot.plot({
      y: {label: "", domain: [0, 100]},
      x: {label: null},
      marks: [
        Plot.barY([{index: "Dissimilarity", value: percentD}], {
          x: "index",
          y: "value",
          fill: "steelblue"
        }),
        Plot.text([{index: "Dissimilarity", value: percentD}], {
          x: "index",
          y: "value",
          text: d => `${d.value}`,
          dy: -6,
          fontWeight: "bold",
          fill: "black"
        })
      ],
      style: {fontSize: "12px"},
      height: 300 // placeholder for now
    })

    const chartDiv = container.querySelector("#chart")
    chartDiv.innerHTML = ""
    chartDiv.appendChild(plot)

    // Adjust the height of the plot to match the height of the sliders column
    const slidersHeight = container.querySelector("#sliders-column").offsetHeight
    chartDiv.style.height = `${slidersHeight}px`
  }

  return container
}
